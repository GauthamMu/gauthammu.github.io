HTML

HTML stands for Hyper Text Markup Language. That is supposed to mean that HTML is not a programming language, but instead a language for displaying things on the web. HTML file names end with either html or htm.

CSS

CSS stands for Cascading Style Sheet. That is supposed to mean that in CSS, the properties of the parent element are taken over by the child. That means that you can give the parent the property to automatically give it to the child too. CSS file names end with css.

JavaScript

Unlike the name suggests, JavaScript and Java are totally different. JavaScript is actually called ECMAScript. JavaScript is the language to actually code in the web. JavaScript file names end with js.

Paper.io

Imagine a snake, which always gets longer. That snake does not actually move, the head of it just gets extended. If anything, including the snake itself, runs into its tail, it dies. But there a safe circle, where it is allowed to run into its tail. It starts in the middle of the safe circle and ollows the cursor of the computer mouse, or the finger on mobile devices. If it goes out of the safe circle and back in, the area between the safe circle and the tail becomes a part of the safe circle, making the safe circle bigger. The snake can run into the safe circles of other players, making the captured area part of the capturing players safe circle. You win if you capture the entire area of the map, which is circular.
